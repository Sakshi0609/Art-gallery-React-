# 🎨 Art Gallery Website (React)

A modern and responsive **Art Gallery web application** built using **React.js**.
This project showcases artworks, artists, and provides an interactive user experience with search and filtering features.

---

## 🚀 Features

* 🏠 Home page with hero section and featured artworks
* 🖼️ Gallery page with:

  * Category-based filtering (Painting, Digital, Photography, etc.)
  * Search functionality
* 👩‍🎨 Artists page with profile cards
* 📩 Contact page with form handling
* 🔗 Navigation using React Router
* 🎨 Clean and responsive UI using Bootstrap

---

## 🛠️ Tech Stack

* React.js (Frontend Library)
* React Router DOM (Routing)
* Bootstrap 5 (Styling)
* CSS3 (Custom Styling)

---
## 🔍 Key Functionalities

### 🔹 Search + Filter (Gallery)

Users can:

* Search artworks by name
* Filter artworks by category
* Combine both search and filter

### 🔹 Routing

* `/` → Home
* `/gallery` → Gallery
* `/artists` → Artists
* `/contact` → Contact

---

## 🎨 UI Highlights
